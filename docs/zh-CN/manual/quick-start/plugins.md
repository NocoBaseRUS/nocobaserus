# 插件

# 什么是插件

- 插件是指遵循一定的规范实现出来的，加载在原系统或平台上，使其能力得到扩展的程序；
- 通常情况下，插件无法脱离原系统或平台而独立运行；
- 可使用不同的插件完成同样的功能，因此插件很容易扩展和替换；

## NocoBase 中的插件

NocoBase 被设计为插件化的架构，除了一个轻量级的内核外，大多数功能都是用插件完成的。

理论上来说，即使是 NocoBase 内置的功能，同样可以通过新的插件来替换。

## 插件管理器

NocoBase 提供了可视化的插件管理器，可以启用、禁用插件，或者对插件进行配置。

插件管理器还处于早期阶段，未来希望我们可以在这里找到各种各样的插件来满足需求。
