# 概述

## 基本概念

NocoBase 的核心是一个基于微内核、插件化设计的开发框架。在此基础上天热的支持任意的扩展开发。

## 特性

## 扩展能力

基于 NocoBase 的微内核设计，在很多模块都考虑相应合理的可扩展性，并向外部暴露了一些扩展点。例如：

* 数据库字段类型
* 已有数据库表的字段
* 通用操作和对特定的资源的操作
* 针对资源和操作的预处理和后处理
* 字段的前端展示组件
* 页面的区块类型
* 用户插件的登入认证方法
* 工作流插件的节点类型和触发类型
* ……

在具体业务中可以根据需求基于以上扩展点进行相应的扩展。
