# 学习路线指南

## 1. 从安装 NocoBase 开始

NocoBase 支持三种安装方式：

- Docker 安装（推荐）
- create-nocobase-app 安装
- Git 源码安装

## 2. 了解 NocoBase CLI

NocoBase CLI 旨在帮助你开发、构建和部署 NocoBase 应用。

最常用的命令包括：

- `yarn nocobase install`
- `yarn nocobase upgrade`
- `yarn dev`
- `yarn test`
- `yarn start`
- `yarn build`
- `yarn pm <method> <plugin>`

## 3. 了解插件管理器

```bash
yarn pm <method> <plugin>
```
