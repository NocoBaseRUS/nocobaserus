# 如何编写测试 <Badge>未实现</Badge>

NocoBase 提供了 @nocobase/test 用于编写和调试插件。

```ts
import { mockServer } from '@nocobase/test';

describe('test', () => {
  let api;

  beforeEach(async () => {
    api = mockServer({});
    await api.database.sync();
  });

  afterEach(async () => {
    await api.database.close();
  });

  it('test resource', () => {
    await app.resource('demos').get();
  });

  it('test request', () => {
    await app.request().get('/');
  });
});
```
