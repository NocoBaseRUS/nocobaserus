# assignFieldValuesForm:configureFields

## 介绍

和字段赋值相关，表示的是字段赋值表单中的配置字段按钮。

## 位置

该按钮分别位于系统中的以下位置：

1. 表单区块中，通过提交按钮的字段赋值配置项打开的字段赋值表单中。
2. 工作流人工节点的操作界面弹窗中，通过表单的继续流程按钮的字段赋值配置项打开的字段赋值表单中。

## 默认分组

### configureFields

- Type: `itemGroup`

当前表的字段列表。

### parentCollectionFields

- Type: `itemGroup`

父表的字段列表。
