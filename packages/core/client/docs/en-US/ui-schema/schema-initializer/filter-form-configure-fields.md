# filterForm:configureFields

## 介绍

表示的是筛选表单中的配置字段按钮。

## 位置

该按钮分别位于系统中的以下位置：

1. 页面中的筛选表单。
2. 弹窗中的筛选表单。

## 默认分组

### displayFields

- Type: `itemGroup`

当前表的字段列表。

### parentCollectionFields

- Type: `itemGroup`

父表的字段列表。

### associationFields

- Type: `itemGroup`

当前表的关系字段列表。
