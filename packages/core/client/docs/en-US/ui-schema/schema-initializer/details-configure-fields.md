# details:configureFields

## 介绍

表示的是详情区块中的配置字段按钮。

## 位置

该按钮分别位于系统中的以下位置：

1. 页面中的详情区块中。
2. 点击查看、编辑、打开弹窗等按钮后，打开的弹窗中的详情区块中。

## 默认分组

### displayFields

- Type: `itemGroup`

当前表的字段列表。

### parentCollectionFields

- Type: `itemGroup`

父表的字段列表。

### associationFields

- Type: `itemGroup`

当前表的关系字段列表。
