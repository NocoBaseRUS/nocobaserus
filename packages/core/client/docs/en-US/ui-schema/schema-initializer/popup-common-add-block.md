# popup:common:addBlock

## 介绍

表示的是一个用于创建区块的按钮，其应用在多个地方，如查看弹窗、编辑弹窗、弹窗按钮打开的弹窗。

## 位置

该按钮分别位于系统中的以下位置：

1. 查看按钮打开的弹窗中。
2. 编辑按钮打开的弹窗中。
3. 弹窗按钮打开的弹窗中。

## 默认分组

### dataBlocks

- Type: `itemGroup`

数据区块列表。

### filterBlocks

- Type: `itemGroup`

筛选区块列表。

### otherBlocks

- Type: `itemGroup`

其它区块列表。
